# External APIs

Это центральное место для заметок по внешним API, которые мы используем в проекте.

Что здесь хранить:

- базовый URL и ссылку на официальную документацию
- какой доступ у нас уже подтвержден
- как авторизоваться
- какие сущности и endpoints уже проверены
- известные ограничения и рабочие гипотезы

Текущие интеграции:

- [`Yandex Metrica`](./yandex_metrica.md)
- [`Yandex Direct`](./yandex_direct.md)
- [`Yandex Tickets`](./yandex_tickets.md)
- [`amoCRM`](./amocrm.md)
- [`Airtable`](./airtable.md)
- [`Fact Layer Source Access`](../fact_layer_source_access.md)

Принцип:

- здесь держим рабочую документацию и способы подключения
- чувствительные секреты лучше хранить вне git и вне общих markdown-файлов
- если в проекте уже есть старые точечные заметки, этот каталог считается более удобной “точкой входа”
