## Notes

- `Sites / points` means different things depending on format:
  - for networks, it is number of venues/locations
  - for single immersive worlds, it can mean rooms, trails, or map points inside one venue
  - for route-based formats, it usually means one route rather than a fixed site network
- `Sqm per participant` is only shown where the metric is meaningful enough to estimate.
- For route-based city formats like `Call Cutta` and `Remote X`, static area is not a useful metric.
- For very large free-roam worlds like `Meow Wolf` or `Sleep No More`, concurrent capacity is often not publicly disclosed, so only event/buyout or careful secondary estimates are used.
